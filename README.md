# 🌍 Buscador de Países Pro

Este es un proyecto educativo desarrollado en **Python** que consume la API de [REST Countries](https://restcountries.com/) para obtener información detallada sobre cualquier nación del mundo en tiempo real.

## 🚀 Características
* 🔍 **Búsqueda interactiva:** Encuentra países por su nombre (inglés o español).
* 💰 **Datos financieros:** Muestra la moneda local con su nombre y símbolo.
* 🗣️ **Cultura:** Lista los idiomas oficiales del país.
* 🚩 **Identidad:** Incluye un enlace directo a la bandera oficial en alta resolución.
* 🗺️ **Geografía:** Detalla la capital, región y subregión.

## 🛠️ Tecnologías utilizadas
* **Python 3.x**
* **Librería Requests:** Para el manejo de peticiones HTTP a la API.
